# Linux权限

```xml
++++对于文件夹下的rwx权限
	r：可以ls，将目录的内容列举出来；如果文件夹没有读权限，只是不能列举出文件夹下的所有文件，其他的查看文件是可以的，如果登录的用户有该文件夹下文件的执行权限；
	w：表示可以在该目录创建或者修改文件；如果没有w（写权限），则不可以删除文件夹下的文件或在文件夹中新建文件；
	x：表示可以进入到该目录，即cd命令。
	
+++++对于文件的rwx权限
	r：可以读，即可以使用cat命令；
	w：可以写，即可以使用vi、vim和echo命令；
	x：表示命令可以被bash解析并执行，允许bash程序去执行文件。
```

对于文件夹下的rwx权限：

+ ***r*** :可以ls，将目录的内容列举出来；如果文件夹没有读权限，只是不能列举出文件夹下的所有文件，其他的查看文件是可以的，如果登录的用户有该文件夹下文件的执行权限。
+ ***w*** :表示可以在该目录创建或者修改文件；如果没有w（写权限），则不可以删除文件夹下的文件或在文件夹中新建文件
+ ***x*** :表示可以进入到该目录，即cd命令；
